#Dotfiles

These are my personal dotfiles for configuring my environments

##Powerline

Please note that my .vimrc and .tmux.conf assume that [powerline](https://github.com/powerline/powerline) is installed

##Syntastic

Make sure that you have the tools used by syntastic for the language you are
working in. For more information
check [the syntastic repo](https://github.com/scrooloose/syntastic)
